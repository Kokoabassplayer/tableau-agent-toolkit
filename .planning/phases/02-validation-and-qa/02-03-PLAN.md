    For src/tableau_agent_toolkit/qa/checker.py:
    - Create StaticQAChecker class with the following methods:

    1. check_all(self, twb_path: Path) -> list[QACheckResult]: Returns results from all checks:
       - check_duplicate_worksheet_names(twb_path)
       - check_unused_datasources(twb_path)
       - check_empty_dashboards(twb_path)
       - check_orphaned_calculations(twb_path)
       - check_missing_workbook_name(twb_path)
       - check_sandbox_smoke_test(twb_path)

    2. check_duplicate_worksheet_names(self, twb_path: Path) -> QACheckResult:
       - Parse TWB, extract root.xpath("//worksheet/@name")
       - If any name appears more than once, FAIL with message "Found duplicate worksheet names: {duplicates}" and details=list of duplicate names
       - Otherwise PASS with "All worksheet names are unique"

    3. check_unused_datasources(self, twb_path: Path) -> QACheckResult:
       - Parse TWB, collect all datasource names: root.xpath("//datasource/@name")
       - Skip datasources named "Parameters" (built-in Tableau parameter datasource)
       - Collect all datasource references from worksheets: look for datasource references in worksheet elements or their children
       - Also check action sources: root.xpath("//action/source/@datasource")
       - If any datasource is never referenced, WARN with "Unused datasources: {names}"
       - Otherwise PASS

    4. check_empty_dashboards(self, twb_path: Path) -> QACheckResult:
       - Parse TWB, find all dashboards: root.xpath("//dashboard")
       - For each dashboard, check if its <zones> child has any <zone> children
       - If any dashboard has empty zones (no zone children or no zones element), FAIL with "Empty dashboards: {names}"
       - Otherwise PASS

    5. check_orphaned_calculations(self, twb_path: Path) -> QACheckResult:
       - Parse TWB, find all calculation columns: root.xpath("//column[calculation]")
       - Extract calculation names (column name attribute)
       - Search for references to each calculation name in:
         - Other calculation formulas: root.xpath("//column/calculation/@formula")
         - Action filters, worksheet table selections (any text content or attribute containing the name)
       - If a calculation is never referenced anywhere, WARN with "Orphaned calculations: {names}"
       - Otherwise PASS
       - Use simple string search (name in text) for reference detection -- do not parse Tableau formula syntax

    6. check_missing_workbook_name(self, twb_path: Path) -> QACheckResult:
       - Parse TWB, check if root element has 'name' attribute and it is non-empty
       - If missing or empty, FAIL with "Workbook name attribute is missing or empty"
       - Otherwise PASS

    7. check_sandbox_smoke_test(self, twb_path: Path) -> QACheckResult:
       - Return QACheckResult(check_name="sandbox_smoke_test", status=CheckStatus.SKIP, message="Sandbox smoke test requires live Tableau Server/Cloud configuration", details=[])
       - This is a stub -- full implementation requires tableauserverclient + server config
