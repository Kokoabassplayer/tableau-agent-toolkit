    For src/tableau_agent_toolkit/validation/semantic.py:
    - Create SemanticValidator class with a single public method: validate(twb_path: Path) -> SemanticValidationResult
    - The validate method:
      1. Parse the TWB file using secure parser: etree.XMLParser(resolve_entities=False, no_network=True)
      2. Get root element via etree.parse()
      3. Collect defined names:
         - datasources: set of root.xpath("//datasource/@name")
         - worksheets: set of root.xpath("//worksheet/@name")
         - dashboards: set of root.xpath("//dashboard/@name")
         - For calculations: root.xpath("//column/calculation/..") then extract parent column name attribute, store in calc_columns set
      4. Check 1 - Broken sheet references: Iterate root.xpath("//dashboard/zones/zone[@name]"). For each zone where name attribute is non-empty and name not in worksheets AND name not in dashboards, create SemanticIssue with severity=ERROR, category="broken_sheet_reference", message="Dashboard zone '{name}' references undefined worksheet or dashboard"
      5. Check 2 - Broken action targets: Iterate root.xpath("//action/source[@worksheet]"). For each source where worksheet attribute is non-empty and worksheet not in worksheets, create SemanticIssue with severity=ERROR, category="broken_action_target", message="Action source references undefined worksheet '{worksheet}'"
      6. Check 3 - Dangling datasource references: Iterate root.xpath("//worksheet"). For each worksheet with a datasource attribute (check via ws.get("datasource")), if datasource value is non-empty and not in datasources set, create SemanticIssue with severity=WARNING, category="dangling_datasource_ref", message="Worksheet '{ws_name}' references undefined datasource '{ds_ref}'"
      7. Check 4 - Dangling field references in calculations: For each column that has a calculation child (root.xpath("//column[calculation]")), extract the formula attribute. Use regex `re.findall(r'\[([^\]]+)\]', formula)` to extract field names. For each field name, strip brackets if present, check if a column with that name exists (compare against all column names in root.xpath("//column/@name") with brackets stripped). If not found, create SemanticIssue with severity=WARNING, category="dangling_field_reference", message="Calculation '{calc_name}' references undefined field '{field_name}'"
      8. Return SemanticValidationResult(valid=len(errors)==0, errors=errors, warnings=warnings)
